# The Tense Times - Языковая Школа

Веб-приложение для языковой школы, где студенты могут изучать иностранные языки.

## Экзаменационный проект по дисциплине "Основы веб-технологий"

### Технологии
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- REST API

### Структура проекта
```
/css        - стили приложения
/js         - JavaScript файлы
/images     - изображения и ресурсы
index.html  - главная страница
profile.html - личный кабинет
```

### Запуск проекта

Для корректной работы необходимо запустить локальный сервер:

```bash
# Python 3
python3 -m http.server 8000

# Или Node.js
npx http-server -p 8000
```

Затем откройте в браузере: http://localhost:8000

### Переключение между Mock-данными и Real API

Проект поддерживает работу с mock-данными для тестирования и с реальным API.

**По умолчанию используется:** Real API (`USE_MOCK_DATA = false`)

**Чтобы переключиться между режимами нужно:**
* В файле `js/mockData.js` изменить строку `const USE_MOCK_DATA = false;` 
   - `false` - использовать реальное API
   - `true` - использовать mock-данные для тестирования

**Mock-данные включают:**
- 8 курсов различных уровней (русский, английский, немецкий, французский и др.)
- 6 репетиторов
- 5 примеров заявок

**Важно:** После изменения флага просто перезагрузить страницу в браузере. 

### Функциональность

#### Этап 1 (10 баллов) 
- Главная страница с полной вёрсткой
- Загрузка и отображение курсов
- Поиск и фильтрация курсов
- Пагинация

#### Этап 2 (10 баллов) 
- Отображение репетиторов
- Поиск репетиторов
- Форма заявки с валидацией
- Отправка заявок
- Система уведомлений

#### Этап 3 (10 баллов) 
- Личный кабинет
- Просмотр списка заявок
- Детали заявки
- Редактирование заявок
- Удаление заявок
- Поиск и сортировка
