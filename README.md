# The Tense Times - Языковая Школа

Веб-приложение для языковой школы, где студенты могут изучать иностранные языки.

## Экзаменационный проект по дисциплине "Основы веб-технологий"

### Технологии
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- REST API

### Структура проекта
```
/css        - стили приложения
/js         - JavaScript файлы
/images     - изображения и ресурсы
index.html  - главная страница
profile.html - личный кабинет
```

### Запуск проекта

Для корректной работы необходимо запустить локальный сервер:

```bash
# Python 3
python3 -m http.server 8000

# Или Node.js
npx http-server -p 8000
```

Затем откройте в браузере: http://localhost:8000

### Mock-данные

По умолчанию проект использует mock-данные для тестирования.

**Тестовые данные включают:**
- 8 курсов различных уровней (английский, немецкий, французский, испанский, китайский, итальянский)
- 10 репетиторов
- 5 примеров заявок в личном кабинете

Чтобы переключиться на реальное API необходимо:
1. Открыть `js/mockData.js`
2. Изменить `const USE_MOCK_DATA = true;` на `false`

### Функциональность

#### Этап 1 (10 баллов) 
- Главная страница с полной вёрсткой
- Загрузка и отображение курсов
- Поиск и фильтрация курсов
- Пагинация

#### Этап 2 (10 баллов) 
- Отображение репетиторов
- Поиск репетиторов
- Форма заявки с валидацией
- Отправка заявок
- Система уведомлений

#### Этап 3 (10 баллов) 
- Личный кабинет
- Просмотр списка заявок
- Детали заявки
- Редактирование заявок
- Удаление заявок
- Поиск и сортировка
